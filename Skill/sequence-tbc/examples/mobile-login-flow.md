## 🧭 สรุปแต่ละช่วง
- **① เข้าใช้งานครั้งแรก** — กรอก OA Number → ระบบตรวจกับ Backend → ส่ง OTP → ยืนยัน OTP → ตั้ง PIN 6 หลัก → เข้าหน้าหลัก
- **② ผู้ใช้เดิมกลับเข้าระบบ** — ตรวจ Session Token; ถ้ายังไม่หมดอายุใช้ PIN เข้าได้เลย (PIN ผิดครบ 7 ครั้ง → ถูกล็อก)
- **③ ล็อกหน้าจอเมื่อไม่ใช้งาน** — ไม่ใช้งานครบ 15 นาที ระบบขอ PIN ซ้ำ
- **④ ลืม PIN / PIN ถูกล็อก** — ยืนยันด้วย OTP เพื่อตั้ง PIN ใหม่

## ⚠ เงื่อนไขสำคัญ
- OTP มีวันหมดอายุ — หมดอายุต้องขอใหม่
- PIN ผิดครบ **7 ครั้ง** จะถูกล็อก และต้องไปขั้นตอนลืม PIN
- Session Token หมดอายุ = ต้องยืนยันตัวใหม่ด้วย OTP
